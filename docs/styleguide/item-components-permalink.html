3.1 #Components.Permalink The Permalink

Toggle example guides Toggle HTML markup

The Permalink Component is added to the end of a heading of a section of a webpage. It is especially helpful in a documentation system, style guide, or pattern library. This Component is an internal link, and makes sections within a document easy to access.

Use with this project’s .h-Hidden and Xmeter’s .h-Inline Helpers.

See the [xPermalink rendering function](../api/global.html#xPermalink_renderer) for usage.

Read https://github.com/chharvey/chharvey.github.io/wiki/Accessible-Skip-Links#internal-permalinks for more info.

DOM:

const {xPermalink} = require('aria-patterns')
document.querySelector('#section-title > h1').append(
  xPermalink.render({ id: 'section-title' })
)
