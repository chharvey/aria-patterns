2.2 #Objects.Navbar The Navbar Object

Toggle example guides Toggle HTML markup

The Navbar Object is a navigation bar whose items each contain a hyperlink and an optional submenu. A Navbar is restricted to items of links and submenus only (as opposed to inputs and/or buttons), where the submenusâ€™ items also contain links. Its purpose is strictly for navigation. This Object is optimized for only 1 nested level of submenus. Further nesting is permitted, but has not been tested in this module.

Note: Enable Javascript to view effects. Ensure the [defer] attribute is set on the <script> (or else move the <script> to the bottom of the <body>).

The navbar may have a horizontal or vertical orientation, specified by the [aria-orientation] attribute. This attribute semantically indicates the navbar layout to assistive technology, as well as defines keyboard interaction. This attribute can be changed dynamically after load. If the author chooses to use a vertical layout, they are responsible for visually styling the navbar in a vertical orientation (specifically, providing a height), as there are no default styles for said layout.
